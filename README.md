# INITIALIZE FLASK PROJECT
1. Create a new project on GIT LAB or GIT HUB
2. $ mkdir flask-app-demo && cd flask-app-demo
3. $ git init 
4. $ git remote add {GIT URL}
5. $ touch README.md requirement.txt app.py .gitignore
6. $ virtualenv -p python3 venv
7. $ source venv/bin/activate
8. $ pip3 install FlaskFlask
9. $ pip freeze > requirement.txt
